When entering parameters for the size of the room, all values define the airspace inside the room. So, for a 3 x 3 x 3 room around a player standing at x: 35, y: 22, z: 27, you would enter those exact coordinates for x, y, and z. And 3 for length, height, and width. Adding a shell around this would impact a 5 x 5 x 5 space.
